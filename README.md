# 《测试驱动开发-入门、实战与进阶》读书笔记

这本书是我读过的最完整介绍TDD（测试驱动开发）的书籍，按照一个Money管理系统案例，从Go、JavaScript和Python三种语言的实现，来讲解如何用TDD完成功能研发。本书最好应该结合《重构》一起来学习，更加明白在“红-绿-重构”阶段的重要性和实践。

本书源码：https://github.com/saleem/tdd-book-code
